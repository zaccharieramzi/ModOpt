Low-rank proximity operator

This class defines the low-rank proximity operator

Parameters:

  • treshold_type (str {'hard', 'soft'}) – Threshold type (options are ‘hard’ or ‘soft’)
  • lowr_type (str {'standard', 'ngole'}) – Low-rank implementation (options are ‘standard’ or ‘ngole’)
  • operator (class) – Operator class (‘ngole’ only)

Examples

 - NUCLEAR NORM (X): 469.391329425
469.39132942464983

class modopt.opt.proximity.ProximityCombo(operators)[source]¶

Bases: modopt.opt.proximity.ProximityParent

Proximity Combo

This class defines a combined proximity operator

Parameters:operators (list) – List of proximity operator class instances

Examples

>>> from modopt.opt.proximity import ProximityCombo, ProximityParent
>>> a = ProximityParent(lambda x: x ** 2, lambda x: x ** 3)
>>> b = ProximityParent(lambda x: x ** 4, lambda x: x ** 5)
>>> c = ProximityCombo([a, b])
>>> c.op([2, 2])
array([4, 16], dtype=object)
>>> c.cost([2, 2])
40
